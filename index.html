IMU-Based Balancing Table (2DOF Control System)

Author: Your Name

Description: This project demonstrates a 2DOF balancing table controlled using an IMU sensor. It utilizes PID control to stabilize the platform in real-time.

Technical Details

  • Microcontroller: Arduino / ESP32
  • IMU Sensor: MPU6050 / BNO055
  • Control Algorithm: PID Controller
  • Actuators: Servo Motors / Stepper Motors
  • Programming Language: C++ (Arduino IDE / PlatformIO)

Challenges & Solutions

One challenge in this project was noise filtering from the IMU sensor. To mitigate this, a complementary filter was used to improve stability.
